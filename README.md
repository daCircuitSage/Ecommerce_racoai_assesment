# DRF Ecommerce API Documentation

This documentation describes the Django REST Framework-based ecommerce API implemented in `ecommerceBackend`.

This repository contains:

- Multiple Django apps for account, categories, products, cart, orders, and payments.
- JWT authentication via `djangorestframework-simplejwt`.
- Stripe payment integration and a strategy pattern for payment providers.
- Docker and Docker Compose configuration for local deployment.
- WhiteNoise static file support for production-like container deployment.

Use the other documents in this folder for detailed architecture, API, deployment, and security information.
